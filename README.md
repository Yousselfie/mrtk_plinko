# MRTK Plinko

Mixed Reality Plinko game targeting the Hololens 2, utilizing MRTK3.
Authors: Youssef Bahloul, Ethan Leyden, Erick Zheng, Kush Patel, Daniel Gubay, DaJuan Hackett, and Dawit Tekeste
